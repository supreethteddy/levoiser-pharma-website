# Levoiser Laboratories Pvt Ltd - Modern Pharmaceutical Website

A modern, responsive pharmaceutical company website built with Next.js, Tailwind CSS, and Framer Motion, inspired by Mankind Pharma's design principles.

## üöÄ Features

- **Responsive Design**: Fully responsive across desktop, tablet, and mobile devices
- **Smooth Animations**: Powered by Framer Motion for elegant transitions and interactions
- **Modern UI**: Clean corporate design with pharmaceutical-inspired color palette
- **Accessibility**: ARIA labels, proper color contrast, and keyboard navigation
- **Performance**: Built with Next.js for optimal loading and SEO

## üé® Design Highlights

- **Animated Header**: Sticky navigation with shrink effect on scroll
- **Hero Section**: Fade-in animations with floating pharmaceutical icons
- **Product Grid**: Hover animations with scale and shadow effects
- **Smooth Scroll**: Intersection Observer for section transitions
- **Interactive Footer**: Animated social icons and back-to-top button

## üõ† Tech Stack

- **Framework**: Next.js 14
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **TypeScript**: Full type safety

## üì¶ Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd PharmaWebsite
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
